# usinggitinbasesas
Examples of GIT Functions in Base SAS
- git_init()
- git_clone()
- git_index_add() (Stage)
- git_commit(), git_commit_log(), git_commit_get()
- git_stage_commit_recursive_macro.sas (reads directories and subdirectories and stages the *.sas files found)
- git_push()
- git_pull()

Advanced Examples: 
- git_branch_new()
- git_branch_chkout()
- git_rebase(), git_rebase_op() in case there are conflicts
- git_stash(), git_stash_pop(), git_stash_apply(), git_stash_drop()
